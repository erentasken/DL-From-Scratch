## [0.1.0] - 2026-02-02

### Added
- Convolutional Neural Network implementation
  - Conv layer with stride and padding
  - Pooling layers (max and average)
  - Flatten layer
  - Fully connected layers
  - ReLU and SoftMax activations
  
- Feed-Forward Neural Network
  - Multi-layer perceptron architecture
  - Flexible layer stacking
  - Training and testing utilities
  
- Optimization algorithms
  - SGD (Stochastic Gradient Descent)
  - SGD with Momentum
  - Adam optimizer
  - Cross-entropy loss
  
- NumPy pattern generation
  - Checkerboard patterns
  - Circle patterns
  - Spectrum patterns
  - Data generators
  
- Regularization techniques
  - Dropout implementation
  - L2 regularization
  
- Test suites
  - Comprehensive unit tests for all modules
  - Gradient checking
  - Integration tests

### Features
- Weight initialization strategies (He, Xavier, Constant)
- Automatic gradient computation
- Loss tracking during training
- Model serialization capabilities
